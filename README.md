## Features

- Add, edit, and delete tasks
- Track task status and deadlines

## How to Run

1. Clone this repo
2. Configure your MySQL credentials in `src/main/resources/application.properties`
3. Build and run: `mvn spring-boot:run` or `mvn clean package && java -jar target/*.jar`
4. Visit [http://localhost:8080](http://localhost:8080)

## Tech Stack

- Java + Spring Boot
- MySQL
- HTML, CSS, JavaScript
